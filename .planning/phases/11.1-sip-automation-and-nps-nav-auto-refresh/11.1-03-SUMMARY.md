# Phase 11.1 Plan 03: NPS NAV Service and Scheme Picker Summary

## What was built

**One-liner:** NPS NAV auto-refresh cloning AMFINavService for npsnav.in latest-min JSON with Decimal-safe Number parser and a searchable scheme picker mirroring AMFISchemePickerView.

### NPSNavService.swift
Full clone of AMFINavService with five swaps:
1. Class/struct name: `NPSNavService` / `NPSScheme`
2. URL: `https://npsnav.in/api/latest-min`
3. UserDefaults key: `"npsNavLastFetchDate"`
4. Parser: `parseNPSLatest(_:)` — JSON array-of-pairs `[[code, Double nav]]` via custom `NPSEntry: Decodable`
5. Asset field: `npsSchemeCode` (was `amfiSchemeCode`)

Identical to AMFINavService: IST daily gate, `isFetching` re-entrancy guard, `Task {}` wrap, `defer { isFetching = false }`, `silently-failing catch`, `currentNAV`/`navAsOfDate` update loop (navAsOfDate set to `todayIST` since latest-min has no per-entry date), `context.save()`.

Key precision decision (T-113-02): JSON Number nav decoded as `Double` intermediate then converted via `Decimal(string: String(format: "%.4f", entry.nav))` — never `Decimal(exactly:)` which is lossy for floating-point.

Defensive parser (Open Question 1): primary decode as bare `[NPSEntry]`; fallback decode as `[String: [NPSEntry]]` for future object-with-data-key wrappers; all failures return `[]` silently (T-113-01).

Public static `historyDateFormatter` (dd-MM-yyyy, IST) added for reuse in SIPAccrualService (PATTERNS.md Cross-Cutting Warning #3 — distinct from AMFINavService's dd-MMM-yyyy formatter).

### NPSSchemePickerView.swift
Mirror of AMFISchemePickerView with three modifications:
1. `var npsNavService: NPSNavService` (was `amfiNavService: AMFINavService`)
2. `[NPSScheme]` (was `[AMFIScheme]`)
3. `filteredSchemes` filters on `name` **OR** `code` (NPS users search by asset-class code)
4. Empty-name fallback: when `scheme.name.isEmpty` (bulk endpoint has no names), the code is shown as the primary row text so every scheme is selectable

All other structure identical: 4-state content switcher, `forceRefresh()` in emptyState, `.frame(minHeight: 44)` touch targets, plain `Text()` throughout (T-113-04), `.searchable(placement: .navigationBarDrawer(displayMode: .always))`, checkmark + tinted background on selected row, `dismiss()` on tap.

### NPSNavServiceTests.swift (10 tests, all passing)
Full replacement of Wave-0 scaffold:
- Fixture test: `latest-min.json` parses to 8 schemes with non-zero Decimal NAVs
- Precision: `[[SM001001, 49.5429]]` → `Decimal(string: "49.5429")` (T-113-02)
- Silent-fail: garbage, empty, object JSON all return `[]`
- IST gate: `shouldFetch` false for today, true for yesterday and distantPast
- Identifiable: `NPSScheme.id == code`
