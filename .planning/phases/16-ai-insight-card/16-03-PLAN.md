## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user category names → model prompt | Category names are user-defined strings injected into the prompt — the only non-numeric user-controlled text in the prompt |
| finance facts → on-device model | Decimal facts injected; never leave the device |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-01 | Information Disclosure | InsightService.generate | mitigate | On-device `LanguageModelSession` only; no `URLSession`, no network entitlement, no telemetry of prompt content. Insight is ephemeral (AI-05) — never persisted or logged in plaintext. |
| T-16-03 | Tampering | InsightPromptBuilder | mitigate | Truncate user-defined category names to ≤30 chars to prevent prompt-stuffing; facts are app-computed Decimals (not user free-text), so injection surface is minimal. |
| T-16-04 | Tampering | session lifecycle | mitigate | Fresh `LanguageModelSession` per generation (no transcript carryover across ranges) — prevents a prior range's context contaminating the new insight. |
| T-16-SC | Tampering | npm/pip/cargo installs | accept | No external package installs (first-party Apple frameworks only). |
</threat_model>
