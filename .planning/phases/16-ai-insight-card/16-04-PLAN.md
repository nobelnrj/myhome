## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| model output → on-screen card | Streamed text crosses into the UI; verified before final display |
| range change → in-flight generation | A stale generation could render against the new range if not cancelled |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-02 | Tampering | AIInsightCard final-display path | mitigate | `InsightVerifier.verify` runs once after the stream completes (Pitfall 8); fallback substituted on `passed == false` so no invented figure reaches the screen (AI-04). |
| T-16-04 | Tampering | .task(id: summary.range) | mitigate | `.task(id:)` auto-cancels the prior generation on range change (D-08); `CancellationError` clears state silently so a stale insight never renders against a new range. |
| T-16-01 | Information Disclosure | AIInsightCard state | mitigate | Insight lives only in `@State`; AI-05 — no persistence, no logging of prompt/output; on-device session only (no network). |
| T-16-05 | Information Disclosure | unavailable-device render | mitigate | `EmptyView()` on every unavailable branch (D-01/SC-2) — no shell text that could leak feature/UX hints or a stuck spinner. |
| T-16-SC | Tampering | npm/pip/cargo installs | accept | No external package installs (first-party Apple frameworks only). |
</threat_model>
