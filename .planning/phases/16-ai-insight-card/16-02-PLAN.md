<objective>
Implement `InsightVerifier` and `InsightFallbackBuilder` so every rupee, percentage
and delta figure in generated insight text is verified against the exact set of
`Decimal` facts pre-computed by `AnalyticsAggregator` (AI-04). Any number the model
invents or mis-rounds triggers substitution of a templated fallback that reads like
a normal terse insight — never an error.

Purpose: AI-04 numeric integrity is the core trust guarantee of the feature — the
model is never allowed to surface a financial figure that Swift did not compute.

Output: `InsightVerifier.verify` + `InsightFallbackBuilder.build` fully implemented;
the four AI-04 tests in `InsightVerifierTests` GREEN.
</objective>
