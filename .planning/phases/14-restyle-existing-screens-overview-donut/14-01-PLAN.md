    - Test: an expense with `isTransfer == true` contributes zero to every category total (self-transfer exclusion); `isTransfer == nil` is included.
    - Test: when fewer than 5 categories have spend, no "Others" entry is emitted.

Return only entries with spent > 0. Write the tests FIRST (RED), then implement (GREEN). Use `import Testing` / `@testable import MyHome` and `private typealias Cat = MyHome.Category` per the OverviewAggregationTests analog.
